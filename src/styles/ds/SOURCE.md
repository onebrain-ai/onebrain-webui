# Vendored OneBrain Design System

These files are a **reference copy** (not an npm dependency) of the canonical
OneBrain Design System — `onebrain-ai/onebrain-design-system`
(`@onebrain/design-system`). Per the WebUI spec decision **D4**, the DS is
consumed as a *two-way living reference*: conform to it here, and when this
surface produces a better pattern, promote it BACK to the DS repo.

- `colors_and_type.css` — tokens (Operator Console palette) + @font-face bindings
- `components.css` — the full component layer (.cyber-*, .btn-*, etc.)
- `fonts/` — Chakra Petch (display) + JetBrains Mono (mono), preserved TTFs

**Do not hand-edit these to diverge.** Re-sync from the DS repo; contribute
improvements upstream. Source captured 2026-06-05.
